Multi-agents event-triggered formation control

Release time：2023-03-14

Introduction

    Multi-Agent Event-Triggered Control (MAETC) algorithms have recently emerged as an effective approach for controlling a group of interconnected agents in a distributed manner. MAETC algorithms can be used in a variety of applications, including robotics, swarm systems, and sensor networks. In MAETC algorithms, each agent communicates with its neighbors and updates its control action only when an event-triggering condition is satisfied. This approach reduces the communication and computation requirements of the agents, which can lead to better performance and scalability compared to traditional control approaches.

    The core idea of MAETC algorithms is to trigger agents' control updates only when significant changes in the system have occurred, rather than continuously updating the control action. An event-triggering condition is usually determined by comparing the current state of an agent to a threshold, and if that condition is satisfied, the agent updates its control action and communicates the change to its neighbors. This event-based approach reduces the amount of information exchanged between agents and can improve the system's overall stability.

    One common type of MAETC algorithm is the distributed consensus algorithm. In this algorithm, each agent updates its control action based on the difference between its current state and the average state of its neighbors. The event-triggering condition is determined by comparing the magnitude of this difference to a threshold. When the threshold is exceeded, the agent updates its control action and continues the consensus process with its neighbors.Another type of MAETC algorithm is the formation control algorithm. In this algorithm, agents work together to maintain a specific formation pattern while avoiding obstacles and collisions. The event-triggering condition is based on the distance between agents in the formation. When a certain distance threshold is exceeded, an agent updates its control action to maintain the desired formation.

    Overall, MAETC algorithms represent a promising approach for controlling groups of interconnected agents in a distributed and scalable manner. The event-triggering approach used in MAETC algorithms reduces the communication and computation requirements of the agents and can lead to improved system performance and stability. These algorithms have a wide range of potential applications and are an active area of research in the field of multi-agent systems.
