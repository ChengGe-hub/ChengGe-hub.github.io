Intelligent control of a car-like vehicle

Release time：2022-10-14

Introduction

    The adaptive dynamic programming (ADP), a neural network (NN) based technology, has aroused great research interests recently. The fundamental principle of this new technique is to improve the control performance by judiciously evaluating system feedback/responses. It can iteratively approximate the optimal control policy and avoid analytically solving the intractable HJB equation.
